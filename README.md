# ionic example

Example app for educational purposes

1. Clone it

	```bash
	git clone git@github.com:anthanh/ionic-example.git
	cd ionic-example
	```

2. Run it

	```bash
	ionic serve --lab
	```

3. Add platforms to project

	```bash
	ionic hooks add
	ionic platform add android
	```

4. Build project into specific platform

	```bash
	ionic build android
	# ionic build ios
	```

5. Emulate project

	```bash
	ionic emulate android
	# ionic emulate ios
	# ionic run android
	```
